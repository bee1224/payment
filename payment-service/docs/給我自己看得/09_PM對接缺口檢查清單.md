# PM 對接缺口檢查清單

> 更新日期：2026-07-09

## 目前需要 PM / 商務確認的事項

| 項目 | 現況 | 需要補充 |
|---|---|---|
| 收款渠道對外公告 | 已縮減為 7 種渠道 | 是否已同步所有商戶與外部文件 |
| 收款欄位使用說明 | 技術文件已改為補充欄位描述 | 是否需要補渠道別填寫範例 |
| 代付審核流程 | 系統已支援 review token | 實際營運審核流程與責任人 |
| callback SLA | 代收商戶 callback 尚未正式重試化 | 是否要求補重試與告警 |

## 已不再列為缺口

以下舊項目因產品範圍已調整，已不再列為缺口：

- 收款 `expired`、`qr_url`、`alipay_qrcode`、`rate` 尚缺穩定來源
- 支付寶相關回應欄位
- 已移除收款渠道的對接承諾
